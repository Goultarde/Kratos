## Summary
Copy a file from source to destination.

## Arguments
| Name | Required | Description |
|------|----------|-------------|
| `source` | Yes | Source file path |
| `destination` | Yes | Destination file path |

## Usage
```
cp C:\secret.txt C:\Windows\Temp\secret.txt
```
