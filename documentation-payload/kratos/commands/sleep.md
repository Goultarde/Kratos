## Summary
Change the agent sleep interval and jitter.

## Arguments
| Name | Required | Description |
|------|----------|-------------|
| `interval` | Yes | Sleep time in seconds |
| `jitter` | No | Jitter percentage (0-100) |

## Usage
```
sleep 30
sleep 60 20
```
